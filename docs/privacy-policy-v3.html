01
Overview

ScoreCalci is a sports scoring and tournament management application. We collect only the data you actively create inside the app. We do not collect any personal, behavioural, or device-level data.

No account creation required. No email. No phone number. No password. Authentication is fully anonymous — you're never asked to identify yourself.

02
Data We Collect

ScoreCalci collects only what you explicitly create:

  • Match records — scores, sport type, date, players or teams you enter, and match duration.
  • Tournament records — bracket data, round results, and settings you configure.
  • Player names — names you enter for saved players. No photos, bios, or contact details.
  • App preferences — theme and display settings stored locally on your device.

We do not collect your name, email address, phone number, location, payment information, contacts, photos, browsing history, advertising identifiers, or similar personal identifiers.

03
Authentication

ScoreCalci uses anonymous authentication via Firebase Auth. A random, non-identifiable user ID is generated automatically when you first open the app. No registration or login is required.

Your anonymous UID is a random string — it cannot be linked back to you as an individual. It exists only to scope your cloud data so no other user can access it.

04
How Data Is Stored

Your data is stored in two locations depending on its type:

Data Type Description Location
Match recordsScores, players, sport, result, durationDevice + Cloud
Tournament recordsBrackets, rounds, progressionDevice + Cloud
Player namesSaved player profilesDevice only
App preferencesTheme, display settingsDevice only
Anonymous UIDFirebase Auth identifierCloud only

Device storage uses Android SharedPreferences. Cloud storage uses Google Cloud Firestore, strictly scoped to your anonymous UID.

05
Data Sharing

We do not sell, trade, rent, or share your data with any third parties for marketing or advertising purposes.

Your data is stored in Google Firebase infrastructure and is also governed by Google's Privacy Policy. Firebase is used solely for anonymous authentication and cloud sync — not analytics or advertising.

05b
Backup & Restore

ScoreCalci allows users to export application data for backup purposes and import previously exported backups.

Backup files are stored only where the user chooses to save them. Users are responsible for securing any exported backup files.

06
Data Deletion

You have full control over your data and can delete it at any time:

  • Delete match history from within the app via Settings → Data & Backup → Clear All Match History.
  • Uninstalling ScoreCalci removes all locally stored data immediately.
  • Cloud data is deleted within 30 days of an account deletion request.

Want the full breakdown? Our dedicated page covers retention timelines, in-app steps, and how to request deletion by email.

Children's Privacy

ScoreCalci is not directed to children under 13. We do not knowingly collect personal information from children. If you believe a child has provided data through our app, contact us and we will remove it promptly.
